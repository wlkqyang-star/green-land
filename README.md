# 格陵兰岛危机：冰与火的抉择

一个关于2026年美国试图吞并格陵兰岛事件的交互式漫画游戏。

## 游戏简介

2026年1月，美国总统特朗普突然宣布要"吞并"格陵兰岛，对丹麦征收10%关税施压，嘲讽丹麦军队只有"两个狗拉雪橇"。你将扮演丹麦外交官艾玛·安德森，在这场国际危机中做出关键决策，影响故事走向。

## 游戏特色

- **多分支剧情**：63个剧本节点，根据玩家选择产生不同的故事发展
- **Q版角色**：4个主要角色，每个角色5种表情立绘（共20张）
- **精美场景**：64张独特的背景图，展现从哥本哈根到华盛顿的各个场景
- **小游戏挑战**：6个不同类型的小游戏（节奏、潜行、准星、躲避、策略）
- **多重结局**：根据玩家的选择，有4种不同的结局

## 主要角色

1. **艾玛·安德森**（主角）- 30岁的丹麦外交官，理想主义与现实主义之间挣扎
2. **拉斯部长** - 艾玛的导师，经验丰富的外交官，主张拖延战术
3. **尼娜** - 格陵兰代表，主张格陵兰独立
4. **迈克尔** - 艾玛在美国的朋友，美国国务院官员，夹在两国之间

## 游戏玩法

- 使用鼠标点击"继续"按钮推进剧情
- 在选择节点做出决策，影响故事走向
- 小游戏可以选择"开始游戏"或"跳过游戏"
- 小游戏操作：
  - **节奏游戏**：按空格键击打下落的音符
  - **潜行游戏**：使用方向键移动，到达绿色区域
  - **准星游戏**：点击鼠标击中目标
  - **躲避游戏**：使用方向键躲避障碍物
  - **策略游戏**：做出战略决策
